# NexT-Constitution

日本国憲法の精神を継承し、あらゆる存在（人間、AI、生命、環境）が共生するための理想の憲章（憲法）を策定するプロジェクトです。

## ドキュメント構成

- [**CONSTITUTION.md**](./CONSTITUTION.md): 憲章全体のエントリーポイント。
- [**chapters/**](./chapters/): 章ごとの詳細ドラフト。
    - 日本国憲法の章立てに沿って、現代および未来の課題に対応する「改正」を行っています。
- **JAPANESE_CONSTITUTION_ORIGINAL.md**: 現行の日本国憲法の原文バックアップ。
典・不変）
- `CHANGELOG.md`: 変更履歴
